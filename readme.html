sitiowebWordpress Descripción general Este repositorio contiene una instalación de WordPress utilizada para la creación de una tienda en línea de videojuegos, desarrollada como parte de una práctica de la materia Desarrollo Web. El sitio se ejecuta en un entorno local mediante XAMPP y está construido utilizando el tema Astra, además del maquetador Elementor y los plugins necesarios para habilitar funcionalidad de tienda digital. La finalidad del proyecto es personalizar una plantilla de Astra para transformarla en un catálogo de videojuegos, integrando páginas, productos, secciones visuales y estructura de navegación según los requerimientos de la práctica. Tema utilizado Dentro de la carpeta wp-content/themes/ el tema instalado es: astra Este tema sirve como base del diseño y permite personalización mediante el Customizer y Elementor. Plugins principales El proyecto utiliza los siguientes plugins: Elementor – para la maquetación del contenido. Starter Templates – para importar la plantilla base. WooCommerce – para la creación de la tienda digital. CartFlows – para flujos de compra. Ultimate Addons for Elementor Lite – para widgets adicionales. WooPayments – para configuración de pagos (en caso de habilitarse). WPForms Lite – para formularios. Variation Swatches for WooCommerce – para variaciones visuales. Cart Abandonment Recovery – para recuperación de carritos. Ejecución del sitio en local (Windows + XAMPP) 1. Requisitos Tener instalado XAMPP (Apache y MySQL). Tener instalado Git para clonar el repositorio (opcional). 2. Clonar el proyecto o colocarlo en htdocs Coloca todo el proyecto dentro de: C:\xampp\htdocs\sitiowebWordpress O clónalo directamente: git clone C:\xampp\htdocs\sitiowebWordpress 3. Crear la base de datos Puedes hacerlo desde phpMyAdmin (http://localhost/phpmyadmin/) o desde consola: mysql -u root CREATE DATABASE videojuegos_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci; exit 4. Configurar WordPress Copia el archivo de ejemplo: Copy-Item wp-config-sample.php wp-config.php Luego abre wp-config.php y modifica: DB_NAME DB_USER DB_PASSWORD DB_HOST 5. Importar la base de datos (opcional) Si cuentas con un respaldo .sql, impórtalo así: mysql -u root videojuegos_db < backup.sql 6. Permisos y carpeta uploads Asegura que la carpeta esté creada y con permisos: wp-content/uploads Debe ser escribible por Apache para permitir subir imágenes. 7. Iniciar servicios y abrir el sitio En XAMPP inicia Apache y MySQL. Abre en tu navegador: http://localhost/sitiowebWordpress/
